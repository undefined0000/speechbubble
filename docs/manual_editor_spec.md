# Manual Editor Spec (v2)

作成日: 2026-03-05  
目的: AI自動顔検出ベースから、手動編集ベースに方向転換する。

## 1. 問題設定

- 既存の自動顔認識精度が制作要求を満たさない
- 同人誌制作では「最終見た目の微調整」が必須
- ユーザーが最終配置を主導できるUIが必要

## 2. プロダクト方針

- AI判定を初期導線から外す
- 1画面で「配置・調整・書き出し」を完結
- スマホ操作を優先（タップ/ドラッグ前提）

## 3. 機能要件

- 画像読込（PNG/JPEG/WEBP）
- 吹き出し追加（ellipse / rounded）
- しっぽ先の手動調整
- テキスト追加（吹き出し内/単独）
- オブジェクトの移動・リサイズ
- レイヤー変更
- Undo/Redo
- PNG書き出し
- プロジェクトJSON保存・読込

## 4. 非機能要件

- ブラウザだけで完結（サーバー推論不要）
- 低遅延編集（ローカルCanvas描画）
- Vercel上で安定動作
- 依存最小化（FastAPI + static assets）

## 5. アーキテクチャ

- Backend: FastAPI
  - `/` と `/editor` で HTMLを返す
  - `/assets` で静的JSを配信
  - `/health` で監視
- Frontend: Vanilla JS + Canvas 2D
  - 画像座標系でオブジェクト管理
  - 表示倍率のみスケール変換
  - Export時は原寸再描画

## 6. 旧機能の扱い

- `/v1/process` は 410 Deprecated
- 旧AIコードは互換のため repo に保持（default運用からは除外）

## 7. 将来拡張

- 縦書きレイアウト
- テンプレ吹き出しプリセット
- 複数ページ一括プロジェクト
- オプションとしてAI補助配置を再導入（完全任意）
